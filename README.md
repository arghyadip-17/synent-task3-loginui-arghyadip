# 🔐 Login Page UI

A modern, clean, and responsive **Login Page UI** built using **HTML & CSS**.
Designed with a dark theme, smooth interactions, and real-world UI/UX practices.

---

## 🚀 Features

* ✨ Clean and modern dark UI design
* 📱 Fully responsive (mobile-friendly)
* 🔐 Email & Password input fields
* 🔑 "Forgot Password" link
* 🔘 Styled Sign In button with hover effects
* 🌐 Google Sign-In button (UI only)
* 🎯 Proper spacing, alignment, and typography
* 🎨 Smooth focus and hover interactions
